# Support

Use GitHub Issues for bug reports.

Open a report here:

[New bug report](../../issues/new?template=bug_report.yml)

## Before reporting

Please check:

1. You are using the latest release.
2. You extracted the ZIP before running the tool.
3. The included folders were not moved apart.
4. You can open the Wingfox course normally in your browser.
5. You logged in before starting the download.

## Include in your report

- App version.
- Windows version.
- What you expected to happen.
- What actually happened.
- Screenshot of the error if available.
- Whether the course page opens normally in your browser.

## Do not include

- Wingfox password.
- Browser cookies.
- Access tokens.
- Paid course files.
- Private account information.
