# Wingfox Downloader

Wingfox Downloader is a simple Windows tool for saving local copies of Wingfox course videos that you are authorized to access.

It is intended for users who prefer local course copies, have unstable internet for streaming, or need offline access to courses they already own or can legally access.

## Download

Download the latest release ZIP from the repository's **Releases** page, extract it, and run the downloader application inside the extracted folder.

## How to use

1. Extract the release ZIP.
2. Run `Wingfox Downloader.exe`.
3. Paste your Wingfox course URL when prompted.
4. When the browser opens, log in to your Wingfox account.
5. Return to the downloader window and press Enter after logging in.
6. Wait for the download to finish.
7. Downloaded videos will be saved in the `Downloads` folder.

## Folder layout

After extraction, the release folder should look similar to this:

```text
Wingfox Downloader/
  Wingfox Downloader.exe
  Downloads/
  bin/
    x64/
      Release/
        net9.0/
          win-x64/
            publish/
              PolyVGet.exe
              other required PolyVGet files
```

Keep the included folders together. Moving only the application file without its required engine files may prevent downloads from working.

## Allowed use

Use this tool only for Wingfox courses you own or are otherwise authorized to access.

Do not use this tool to copy, share, sell, upload, mirror, or redistribute Wingfox courses or course files.
